## 🔍 STEP별 상세 진단

### STEP 1: 글로벌 서비스 이해하기

#### 1.1 GlobalBenefits (글로벌 시장 통계)

**현재 상태**:
```
| 32      | 2       | 45      |
| 배      | 배      | 개      |
|         |         | 국      |
```

**문제점**:
- ❌ `text-3xl sm:text-4xl` 숫자 크기가 모바일에서 과도하게 큼
- ❌ "개국", "배" 단위가 숫자와 분리되어 별도 줄로 내려감
- ❌ 카드 내 텍스트가 세로로 길어져 가독성 저하

**개선안**:
```tsx
// AS-IS
<div className="text-3xl sm:text-4xl font-black">
  <AnimatedNumber value={32} suffix="배" />
</div>

// TO-BE
<div className="text-2xl sm:text-3xl font-black whitespace-nowrap">
  <AnimatedNumber value={32} suffix="배" />
</div>
<div className="text-[10px] sm:text-xs text-idus-black-50">
  {stat.subtext}
</div>
```

**세부 변경사항**:
| 속성 | 현재 | 개선 |
|------|------|------|
| 숫자 크기 | `text-3xl sm:text-4xl` | `text-2xl sm:text-3xl` |
| 단위 표시 | 숫자와 단위 줄바꿈 | `whitespace-nowrap` |
| 설명 텍스트 | `text-xs` | `text-[10px] sm:text-xs` |
| 아이콘 크기 | `w-12 h-12` | `w-10 h-10 sm:w-12 sm:h-12` |

**현재 상태**: ✅ 양호

### STEP 2: 판매 준비하기

```tsx
      <X className="w-3 h-3 text-red-500 flex-shrink-0" />

    </div>
  ))}
</div>
```
