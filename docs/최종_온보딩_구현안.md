# 🌏 글로벌 작가 온보딩 페이지 최종 구현안 (v4.0)

## 📋 프로젝트 개요

### 🎯 핵심 목적
**KR 서버에 입점된 작가님들이 글로벌 판매에 대한 어려움을 낮추고, 
재미있고 쉬운 온보딩을 통해 실제 글로벌 작가 등록까지 이어지도록 유도**

### 핵심 원칙
- ✅ **등록 전환 최적화**: 학습 → 실제 등록까지 자연스러운 유도
- ✅ **자격 사전 확인**: 판매 금지 카테고리, 사업자 등록 조건 필터링
- ✅ **데이터 수집**: 작가 정보 수집 → 구글 시트 아카이빙
- ✅ **보상 시스템**: 완료 시 KR 광고포인트 1만점 지급
- ✅ **2026 확장 대비**: 식품/디지털 카테고리 별도 리스트 수집

### 크롤링 데이터 기반
- **원본 노션**: https://idus.notion.site/d50ff6b383014f1abfed25ccab054007
- **크롤링 문서**: `./노션_크롤링_데이터.md` (완전판)
- **캡처 이미지**: `./screenshots/` (시각 자료)

```typescript
interface OnboardingData {
  // 기본 정보
  artistName: string;           // 작가명/브랜드명
  phoneNumber: string;          // 연락처
  
  // 자격 조건
  hasBusinessNumber: boolean;   // 사업자등록번호 보유 여부
  categories: string[];         // 판매 카테고리 (복수 선택)
  
  // 2026 확장 카테고리
  interestedIn2026: {
    food: boolean;              // 식품
    digital: boolean;           // 디지털 작품
  };
  
  // 학습 진행
  qualificationStatus: 'qualified' | 'no_business' | 'restricted_category';
  learningProgress: {
    step1Completed: boolean;
    step2Completed: boolean;
    step3Completed: boolean;
    completedAt?: Date;
    totalTimeMinutes?: number;
  };
  
  // 등록 전환
  registrationClicked: boolean;
  registrationCompleted?: boolean;
  
  // 메타데이터
  createdAt: Date;
  updatedAt: Date;
  utmSource?: string;
  utmMedium?: string;
  utmCampaign?: string;
}
```
