## 인포그래픽 전환 대상 분류

### 🟢 인포그래픽 전환 권장 (4개)

| 콘텐츠 | 전환 이유 | 인포그래픽 타입 | 우선순위 |
|--------|----------|---------------|---------|
| **global-intro** | 숫자 데이터(32배, 2배, 45개국) 시각화 효과 높음 | 비교 카드 + 숫자 강조 | 🔴 높음 |
| **settlement** | 정산 플로우 + 수수료 비교가 복잡 | 플로우 차트 + 간소화 테이블 | 🟡 중간 |
| **artist-registration** | 등록 경로 3단계가 명확함 | 스텝 플로우 | 🟡 중간 |
| **product-registration** | 설정 4단계가 명확함 | 스텝 체크리스트 | 🟡 중간 |

### 🟡 부분 인포그래픽 전환 (1개)

| 콘텐츠 | 전환 범위 | 인포그래픽 타입 |
|--------|----------|---------------|
| **sales-tips** | 일본 vs 영어권 비교만 | 2컬럼 비교 카드 |

### 🔵 부록 이관 권장 (2개)

| 콘텐츠 | 이관 이유 | 부록 위치 |
|--------|----------|----------|
| **document-guide** (소포수령증) | 판매 후 필요, 초기 학습에 불필요 | 부록 > 증빙서류 |
| **settlement** 수수료 테이블 | 상세 정보, 핵심만 요약 노출 | 부록 > 수수료 상세 |

### ⚪ 현행 유지 (텍스트 기반)

| 콘텐츠 | 유지 이유 |
|--------|----------|
| 자주 묻는 질문 답변 예시 | 텍스트가 더 적합 (복사 활용) |
| 물류창고 주소 | 복사 기능 필요 |

### 🎯 정성적 기대효과

   - 핵심만 빠르게 파악 가능

   - 숫자 데이터의 시각적 임팩트 증가
   - 프로세스/단계의 직관적 이해

   - 깔끔한 디자인으로 진입 장벽 낮춤
   - 완료 가능해 보이는 학습량

4. **부록 활용도 증가**
   - 필요한 정보만 모아둔 실용적 레퍼런스
   - 판매 후에도 재방문 가치 제공
