## 2. 페이지별 브랜드 강화 개선안

- 기본적인 오렌지 컬러 적용
- 배경이 단조로움

#### 개선안

```css

.landing-hero {
  background: 
    linear-gradient(180deg, rgba(255,255,255,0.95) 0%, rgba(245,245,245,0.9) 100%),
    url('/brand/brand assets/pattern01.png');
  background-size: cover, 300px 300px;
```

**적용 요소:**

## 6. 브랜드 가이드라인 요약

### Do's ✅
- 오렌지 컬러는 항상 CTA와 중요 강조에 사용
- 브랜드 아이콘은 감정을 전달할 때 적극 활용
- 패턴은 배경에 10% 이하 opacity로 은은하게
- 그라데이션은 orange → orange-dark 방향 유지

### Don'ts ❌
- 오렌지 색상을 경고/에러에 사용하지 않음 (빨간색 유지)
- 브랜드 아이콘을 너무 작게 사용하지 않음 (최소 20px)
- 패턴을 전면에 강하게 사용하지 않음
- 여러 그라데이션을 혼용하지 않음

## 7. 결론

idus 글로벌 온보딩 페이지는 이미 기본적인 브랜드 컬러와 아이콘을 잘 활용하고 있습니다. 
하지만 **"핸드메이드 전문 플랫폼"**이라는 idus의 정체성을 더욱 강화하기 위해:

1. **오렌지 그라데이션**을 통한 깊이감과 프리미엄 느낌
2. **브랜드 아이콘**을 통한 따뜻하고 친근한 감성
3. **패턴 활용**을 통한 수공예 느낌의 배경
4. **일관된 애니메이션**을 통한 생동감

을 적용하면, 단순한 "시스템 페이지"가 아닌 **"idus만의 특별한 경험"**을 제공할 수 있습니다.
