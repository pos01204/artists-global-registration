## 랜딩 페이지 개선안

### 1. 숫자 카운트업 애니메이션 적용

**현재**: 정적인 숫자 표시
**개선**: 스크롤 진입 시 카운트업 애니메이션

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   ┌─────────────┐  ┌─────────────┐  ┌─────────────┐            │
│   │             │  │             │  │             │            │
│   │  0 → 45개국  │  │  0 → 32배   │  │  0 → 2배    │            │
│   │   ↑ 카운트업 │  │   ↑ 카운트업 │  │   ↑ 카운트업 │            │
│   │             │  │             │  │             │            │
│   │  판매 국가   │  │ 글로벌 시장  │  │ 해외 평균    │            │
│   └─────────────┘  └─────────────┘  └─────────────┘            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**컴포넌트 구조**:
```tsx
// AnimatedStatCard.tsx
interface Props {
  value: number;
  suffix: string;
  label: string;
  icon: React.ReactNode;
  delay?: number;
}

// useInView + useCountUp 조합
// 카드가 화면에 보일 때만 애니메이션 시작
```

### 2. 히어로 섹션 타이핑 애니메이션

**현재**: 정적인 타이틀
**개선**: react-type-animation으로 타이핑 효과

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│            ✨ 전 세계 45개국 판매                               │
│                                                                 │
│         글로벌 작가가 되어보세요!|                              │
│              ↑ 타이핑 애니메이션                                │
│                                                                 │
│         전 세계 고객에게 작품을 선보일 준비가 되셨나요?         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 소셜 프루프 동적 강조

**현재**: 정적인 "3,851명" 표시
**개선**: 실시간 증가 효과 + 펄스 애니메이션

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   ┌────────────────────────────────────────────────────┐       │
│   │  🟢 이미 3,851명 → 3,852명의 작가님이             │       │
│   │     글로벌 판매를 시작했어요                       │       │
│   │     ↑ 숫자 증가 애니메이션 (랜덤 간격)            │       │
│   └────────────────────────────────────────────────────┘       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4. 보상 배너 플로팅 아이콘

**현재**: 정적인 선물 이미지
**개선**: 부드러운 플로팅 + 쉬머 효과

```tsx
// 선물 아이콘 플로팅 애니메이션
<motion.div
  animate={{ 
    y: [0, -10, 0],
    rotate: [0, 5, -5, 0]
  }}
  transition={{ 
    duration: 3,
    repeat: Infinity,
    ease: "easeInOut"
  }}
>
  <Image src="/선물.png" ... />
</motion.div>
```

│   │   ✓     │ ━━━━━> │   📚    │ ━ ─ ─> │   ✈️    │            │
│   │ 자격확인 │        │  학습   │        │  등록   │            │

│                      ↑ 펄스 애니메이션                          │

## 공통 컴포넌트 개선

### 1. Button 컴포넌트 강화

```tsx
// 새로운 variant 추가
type ButtonVariant = 
  | 'primary'      // 기존
  | 'secondary'    // 기존
  | 'gradient'     // NEW: 그라디언트 쉬머
  | 'glow'         // NEW: 글로우 효과
  | 'outline';     // 기존

// gradient variant CSS
.btn-gradient {
  background: linear-gradient(90deg, 
    var(--idus-orange) 0%, 
    #FF8C00 50%, 
    var(--idus-orange) 100%);
  background-size: 200% 100%;
  animation: shimmer 2s ease-in-out infinite;
}
```

### 2. Card 컴포넌트 호버 효과 통일

```tsx
// 호버 시 일관된 효과
.card-hover {
  transition: all 0.3s ease;
}

.card-hover:hover {
  transform: translateY(-4px);
  box-shadow: 0 12px 24px -8px rgba(0, 0, 0, 0.1);
}
```

### 3. 로딩 스피너 브랜드화

```tsx
// 현재: 단순 원형 스피너
// 개선: idus 로고 기반 로딩

<motion.div
  animate={{ rotate: 360 }}
  transition={{ duration: 1, repeat: Infinity, ease: 'linear' }}
>
  <BrandIcon name="best" size={32} />
</motion.div>
```

### 4. 페이지 전환 애니메이션

```tsx
// 페이지 진입 시 일관된 페이드인
// layout.tsx에 적용

<motion.div
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.4 }}
>
  {children}
</motion.div>
```
